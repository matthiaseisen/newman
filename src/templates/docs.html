About

Newman API is a quick and easy way to add form mailing functionality to your (static) website.
The design and complexity of your forms is entirely up to you. You code it, Newman sends it.
JavaScript is not required.


Getting Started

   
    <!-- This is some boilerplate code to get you started.  -->
    
    <form method="post" action="http://newmanapi.com/submit/" >
        
        <!-- These are important! -->
        <input type="hidden" name="newman_api_key" value="YOUR API KEY" />
        <input type="hidden" name="newman_fields" value="field1,field2,field3" />
        
        <!-- These are optional. -->
        <input type="hidden" name="newman_required" value="field1,field3" />
        <input type="hidden" name="newman_thanks_url" value="http://www.example.com/thanks/" />
        <input type="hidden" name="newman_oops_url" value="http://www.example.com/oops/" />
        <input type="hidden" name="newman_error_url" value="http://www.example.com/error/" />

        <!-- These are the actual input fields. -->
        Some Field:<br />
        <input type="text" name="field1" /><br />
        Another Field:<br />
        <input type="text" name="field2" /><br />
        A third Field:<br /> 
        <textarea name="field3" rows="10" cols="35" ></textarea><br />

        <input type="submit" value="submit" />

    </form>

Starting from Scratch

   
    <form method="post" action="http://submit.newmanapi.com/" >
        YOUR CODE HERE
    </form>
     

Setting your API Key

required

Get your API Key from the Account Dashboard.

   
    <input type="hidden" name="newman_api_key" value="YOUR API KEY" />
     

Defining your Fields

required

Newman expects a comma-separated list of your input element's name attributes.
  
    <input type="hidden" name="newman_fields" value="field1,anotherfield,a-third-field" />
     

Making Fields Mandatory

optional

   
    <input type="hidden" name="newman_required" value="field1,field3" />
     

Configuring Redirects

optional

URL Triggered if Default
newman_thanks_url The user has successfully submitted the form http://www.newmanapi.com/thanks/
newman_oops_url The user has ignored a mandatory field http://www.newmanapi.com/oops/
newman_sorry_url The form isn't set up correctly or you've run out of credits http://www.newmanapi.com/sorry/ 
    
    <input type="hidden" name="newman_thanks_url" value="http://www.example.com/thanks/" />
    <input type="hidden" name="newman_oops_url" value="http://www.example.com/oops/" />
    <input type="hidden" name="newman_error_url" value="http://www.example.com/error/" />
